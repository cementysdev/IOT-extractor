# IOT-extractor

A base program to comput data over an excel

## Author

Developed by Thibault HOGUIN.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Edit the device_ids list in main.py.

Run the script:

```bash
python main.py
```
